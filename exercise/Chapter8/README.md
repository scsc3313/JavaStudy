###Chapter 8

- 임의의 정수를 입력시 첫자리부터 정수를 읽어가면서 그 정수와 그 정수의 갯수를 차례로 출력한다.

    input -> 11882736284

    output 128322316141(1 = 2개 8 = 3개 2 = 2개 3 = 1개 6 = 1개 4 = 1개 이런뜻임)



###Solve

    1. 정수를 한자리 정수씩 읽어나가는 것이기 때문에 String 문자열로 입력을 받는다.
    2. String을 char[]로 만들고 char[]를 int[]로 변환하여 카운트를 한다.
    3. for문을 이용하여 출력한다.
